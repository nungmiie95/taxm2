📊

จุดสำคัญ
ต้องดูว่าเงินได้สุทธิอยู่ช่วงใด และคิดทีละช่วงตามตาราง

150,001 – 300,000 บาท5%
300,001 – 500,000 บาท10%

750,001 – 1,000,000 บาท20%
1,000,001 – 2,000,000 บาท25%
2,000,001 – 5,000,000 บาท30%
มากกว่า 5,000,000 บาท35%

ตัวอย่างที่ 1 ระดับง่าย

โจทย์
1
นาย ก. มีเงินได้พึงประเมิน 200,000 บาท
2
ค่าใช้จ่าย 80,000 บาท
3
ค่าลดหย่อน 60,000 บาท
วิธีทำ
เงินได้สุทธิ = 200,000 - 80,000 - 60,000
เงินได้สุทธิ = 60,000 บาท
เทียบตาราง
เงินได้สุทธิ 60,000 บาท อยู่ในช่วง 0 – 150,000 บาท
อัตราภาษีที่ใช้ = ยกเว้น
ภาษีที่ต้องชำระ = 0 บาท

ตัวอย่างที่ 2 ระดับกลาง

โจทย์
1
นาย ค. มีเงินได้พึงประเมิน 360,000 บาท
2
ค่าใช้จ่าย 100,000 บาท
3
ค่าลดหย่อน 80,000 บาท
วิธีทำ
เงินได้สุทธิ = 360,000 - 100,000 - 80,000
เงินได้สุทธิ = 180,000 บาท
เทียบตารางแบบขั้นบันได
0 – 150,000 บาท = ยกเว้น
ส่วนที่เกิน 150,000 บาท คือ 30,000 บาท
30,000 × 5% = 1,500 บาท
อัตราภาษีที่ใช้ = 5%
ภาษีที่ต้องชำระ = 1,500 บาท

ตัวอย่างที่ 3 ระดับสูงขึ้น

โจทย์
1
นาย จ. มีเงินได้พึงประเมิน 520,000 บาท
2
ค่าใช้จ่าย 100,000 บาท
3
ค่าลดหย่อน 90,000 บาท
วิธีทำ
เงินได้สุทธิ = 520,000 - 100,000 - 90,000
เงินได้สุทธิ = 330,000 บาท
เทียบตารางแบบขั้นบันได
0 – 150,000 บาท = ยกเว้น
150,001 – 300,000 บาท = 150,000 × 5% = 7,500 บาท
ส่วนที่เกิน 300,000 บาท คือ 30,000 บาท
30,000 × 10% = 3,000 บาท
รวมภาษี: 7,500 + 3,000 = 10,500 บาท
อัตราภาษีที่ใช้ = 5%, 10%
ภาษีที่ต้องชำระ = 10,500 บาท

ขั้นตอนจำง่าย

📖
1. อ่านโจทย์
ดูให้ครบว่าโจทย์กำหนดเงินได้พึงประเมิน ค่าใช้จ่าย และค่าลดหย่อนเท่าใด
🧮
2. หาเงินได้สุทธิ
ใช้สูตร เงินได้สุทธิ = เงินได้พึงประเมิน - ค่าใช้จ่าย - ค่าลดหย่อน
📊
3. เทียบตารางภาษี
ตรวจว่าจำนวนเงินได้สุทธิอยู่ในช่วงใด และคิดภาษีตามขั้นบันได
💰
4. คำนวณภาษี
คำนวณเฉพาะส่วนที่ต้องเสียภาษี แล้วรวมเป็นภาษีที่ต้องชำระ
